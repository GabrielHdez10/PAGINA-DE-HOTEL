# Documentación del Sistema de Reservación de Hotel

## Objetivo
Gestionar documentación técnica estructurada utilizando Markdown y herramientas de automatización en la nube, asegurando que la arquitectura del sistema sea clara, accesible y fácil de actualizar.

## Descripción
Este sistema permite a los usuarios consultar habitaciones disponibles, realizar reservaciones, gestionar pagos y recibir confirmaciones. También permite al administrador gestionar habitaciones, clientes y reservas.
