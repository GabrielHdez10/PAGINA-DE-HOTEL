# Diccionario de Datos

| Elemento | Tipo | Descripción | Datos |
|---|---|---|---|
| Usuario | Entidad | Cliente que realiza reservas | nombre, correo, teléfono |
| Administrador | Entidad | Controla el sistema | usuario, contraseña |
| Habitación | Entidad | Espacio disponible | número, tipo, precio |
| Reserva | Entidad | Registro de reservación | fecha, cliente, habitación |
| Pago | Proceso | Gestión de cobro | monto, método |
